
# Twitter clone

A small project created as playground to learn react-query


## Run Locally

Clone the project

```bash
  git clone https://github.com/DanielFijolek/twitter_clone
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the json-server

```bash
  json-server --watch db.json --port 3001
```

Start the web-app

```bash
  npm run start
```
